Micosoft Intune mass device identifier add/remove PowerShell Script

Problem: Die IT-Abteilung erhält regelmäßig defekte/veraltete via Intune gemanagte Geräte zurück. Die Geräte werden daraufhin dem remarketing zugeführt. Die IT-Abteilung erhält vom Remarketer eine Excel Liste mit den Seriennummern der Geräte. Diese Geräte müssen aus Sicherheitsgründen nun aus Intune entfernt werden. Intune bietet leider über das frontend nicht die Möglichkeit eine Excel/CSV Liste hochzuladen.

Lösung: Entwicklung eines Powershell-Scripts, welches über die Graph-API eine zu übergegebene Liste von Seriennnummern aus Intune entfernt/hinzufügt.
